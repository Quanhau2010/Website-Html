Welcome to HERLYS API

Một dịch vụ API và miễn phí, cung cấp nhiều tính năng đa dạng cho các idol.

Hướng Dẫn Cho Người Mới:

  1. Chọn danh mục API từ thanh bên
  2. Chọn endpoint mong muốn
  3. Xem tài liệu của endpoint
  4. Gửi yêu cầu GET đến URL của endpoint
  5. Xử lý phản hồi JSON

Điều khoản dịch vụ:

  1. Không gửi quá nhiều yêu cầu.
  2. Chủ sở hữu có thể thay đổi điều khoản của API này bất cứ lúc nào để duy trì chất lượng API.
  3. Xin đừng tấn công nó, điều đó sẽ gây hại.
  4. Chủ sở hữu không chịu trách nhiệm về những gì bạn làm khi sử dụng trang web này.
